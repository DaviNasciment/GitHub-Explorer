# GitHub Explorer React
> Aplicação em React

Ambiente de Produção
  
## Tecnologia

* HTML5
* CSS3
* ReactJS
* Node
* Yarn

## Começando
 
Instalar
>    git clone https://github.com/DaviNasciment/gitHub-explorer-react.git
 
## Instalações
Instalar o node_modules.
>    yarn
>    
buildar a pagina dist.
>    yarn build

## Iniciando 
Starta o servidor
>    yarn dev
