# GitHub Explorer React
> Aplicação em React

[Ambiente de Produção](https://github-explorer-react-d.netlify.app/)
  
## Tecnologia

* HTML5
* CSS3
* ReactJS
* Node
* Yarn

## Começando
 
Instalar
>    git clone https://github.com/DaviNasciment/gitHub-explorer-react.git
 
## Instalações
Instalar o node_modules.
>    yarn install

## Iniciando 
Starta o servidor
>    yarn dev
